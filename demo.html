Surface tint

--surface-hue / --surface-sat tint every --bg* surface from one place (neutral by default). Because the tint vanishes at 100% lightness, a themed site gets "neutral-white chrome over faintly-tinted paper" for free — hue contrast stays visible near white where a luminance step would not.

The Art of Readable Typography

Good typography is invisible. When readers are fully immersed in a text, they don't notice the typeface, the line spacing, or the margins. They simply absorb the content. This is the goal every designer should strive for.

Yet achieving this invisibility requires deliberate, careful choices. Every element of typographic design plays a role in either enhancing or hindering readability. The spacing between paragraphs, the relationship between headings and body text, and the rhythm of the page all contribute to the reading experience.

The Importance of Vertical Rhythm

Vertical rhythm refers to the consistent spacing patterns that guide the eye down the page. When text follows a predictable rhythm, readers can focus on the content rather than consciously parsing the layout.

Consider these principles:

  • Paragraph spacing should relate to line height
  • Headings need more space above than below
  • List items require tighter spacing than paragraphs
  • Block elements like quotes need visual separation

By basing all spacing on a single unit (the line height), we create harmony throughout the document.

Typography is the craft of endowing human language with a durable visual form.

Robert Bringhurst

Practical Application

Implementing vertical rhythm in CSS has become easier with custom properties. We can define a base rhythm unit and derive all other spacing values from it:

:root {
  --prose-line-height: 1.6;
  --prose-rhythm: calc(1rem * var(--prose-line-height));
}

This approach ensures that as the base font size scales with the viewport, all spacing scales proportionally. The result is a design that feels consistent across devices.

Good typography takes time to perfect, but the investment pays dividends in reader engagement and comprehension.
